## Overview

Welcome to Maycon Mota, your go-to platform for creating, organizing, and managing project documentation. You streamline workflows by centralizing all docs in one space, supporting MDX for rich, interactive content. Benefit from version control, collaborative editing, and seamless publishing to keep your teams aligned.

## Key Features

Maycon Mota empowers you with powerful tools for documentation management.

  <Card title="MDX-Powered Pages" href="/docs/mdx-guide" icon="book-open" horizontal="false">

  <Card title="Version Control" href="/docs/versioning" icon="git-branch" horizontal="false">

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus" title-type="p">
    Sign up at `https://mayconmota.com` with your email. Verify your account to unlock full features.
  </Step>

  <Step title="New Workspace" icon="plus" title-type="p">
    Create a new documentation space for your project. Name it and select your primary color `{ #3B82F6 }`.
  </Step>

  <Step title="Add First Page" icon="file-text" title-type="p">
    Generate your `introduction.mdx` using the built-in templates. Customize with components.
  </Step>

  <Step title="Publish" icon="globe" title-type="p">
    Preview and deploy your docs. Share the live URL with your team.
  </Step>
</Steps>

<CodeGroup show-lines="true" tabs={[]}>
  ```javascript
  // Example: Embed Maycon Mota docs in your app
  import { DocsEmbed } from '@mayconmota/embed';

  function App() {
    return (
      <DocsEmbed workspaceId="your-workspace-id" page="introduction" />
    );
  }
  ```

  ```python
  # Example: Fetch latest docs via API
  import requests

  response = requests.get(
    'https://api.mayconmota.com/workspaces/your-workspace/pages/introduction',
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
  )
  print(response.json())
  ```
</CodeGroup>

## Brand Guidelines

Follow these guidelines to maintain consistency across your documentation.

<Callout kind="info" collapsed="false">
  Use the primary brand color `{ #3B82F6 }` for headings, buttons, and accents. Pair it with neutral grays for readability. Avoid using it for text to ensure accessibility.
</Callout>

<Expandable title="Full Color Palette" default-open="false">
  - Primary: `{ #3B82F6 }`

  - Secondary: `{ #1D4ED8 }`

  - Success: `{ #10B981 }`

  - Danger: `{ #EF4444 }`
</Expandable>
