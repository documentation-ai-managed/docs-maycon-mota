## Getting Started

To start documenting your projects with Maycon Mota:

1. Sign in at `https://mayconmota.com`
2. Click "New Project" in the dashboard
3. Enter a name and description
4. Click "Create"

Your project space is now ready.

## Next Steps

- Add pages using the sidebar
- Organize docs in folders
- Collaborate by inviting team members
